# GSM-Facial-Recognition-Door-Access-Control-

## Problem Statement
Current entry control access systems in Uganda are either outdated or expensive to maintain. Physical keys and PINs can be easily compromised while imported biometric systems are costly and lack customization. 
There is a need for an affordable, intelligent and locally made adaptable solution to secure sensitive areas and accurately track, log and control physical access entry to sensitive areas in banks, research facilities, government institutions, labs in universities etc.

## Our project solution
The GSM Facial Recognition Door Access Control provides a modern, efficient, and secure approach to facility access management. By automating identification using facial recognition, the system eliminates the weaknesses of traditional methods such as key duplication or PIN sharing. It enhances safety and accountability through automated entry logs that record names, dates, and times of access entry essential for banks, institutions like research laboratories, universities, government offices, and corporate environments. 
In a practical setting, it can also be able to notify the admin when its disconnected from the internet so that it can be remotely controlled.
